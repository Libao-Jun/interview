### 基础知识

先补足能支撑大多数前端追问的底层知识，共 **15 篇**。

#### Web 核心三件套 + 语言进阶

| 章节 | 核心能力 |
| --- | --- |
| **HTML** | 语义标签、音视频、Canvas/SVG、Web Storage、Web Worker |
| **CSS** | 盒模型、BFC、选择器优先级、Flex/Grid、重排重绘、响应式 |
| **ES6+** | 变量声明、解构、箭头函数、Promise、模块化、迭代器与生成器 |
| **JavaScript** | 数据类型、原型链、闭包、this、事件循环、异步编程 |
| **TypeScript** | 类型体操、泛型约束、条件类型、映射类型、类型守卫 |

#### 框架与状态管理

| 章节 | 核心能力 |
| --- | --- |
| **Vue** | 响应式原理（Proxy vs defineProperty）、组件化、生命周期、编译优化 |
| **Vue Router** | 路由守卫、动态路由、导航解析流程、懒加载策略 |
| **React** | Fiber 架构、Hooks 机制、并发渲染、虚拟 DOM Diff |
| **状态管理** | Vuex/Pinia vs Redux/Zustand、单向数据流、中间件机制 |

#### 浏览器与网络

| 章节 | 核心能力 |
| --- | --- |
| **浏览器** | 渲染管线、V8 引擎、垃圾回收、Web 安全、跨域方案 |
| **Network** | HTTP 演进（1.1/2/3）、TCP/UDP、缓存策略、WebSocket |

#### 跨端开发

| 章节 | 核心能力 |
| --- | --- |
| **H5 公众号** | JSSDK 接入、微信 webview 兼容、授权流程 |

## 推荐阅读路径

1. **先看[刷题策略](/docs/guide)**，明确复习节奏和输出目标 —— 知道"怎么复习"比"复习什么"更重要。
2. **基础知识打底**，补齐 Web 核心 + 浏览器 + 网络等底层机制 —— 面试追问 80% 来自这里。
3. **框架与状态管理进阶**，理解 Vue 和 React 的渲染模型与数据流 —— 区分"用过"和"懂了"。
4. **工程化场景实战**，把项目经验组织成可验证的回答 —— 从性能指标到架构决策，最容易拉开差距。

:::info
每读完一篇，把结论改写成你自己的回答版本。知识库提供的是**框架和线索**，真正的面试输出需要你用自己的语言填入细节。
:::
