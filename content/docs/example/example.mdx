## MDX 插件

- [MDX Plugins](https://www.fumadocs.dev/docs/headless/mdx): MDX 插件

## 提示信息

### 参考文档
- [Fumadocs](https://www.fumadocs.dev/docs/markdown): Markdown文档（MDX 的 JSX 语法）

### 信息类型
- 信息
- 提示
- 警告
- 成功
- 想法

### Example

```tsx
<Callout>Hello World</Callout>

<Callout title="Title" type="warn">
  Hello World
</Callout>

<Callout title="Title" type="error">
  Hello World
</Callout>

<Callout title="Title" type="idea">
  Hello World
</Callout>
```

### 案例效果
<Callout>Hello World</Callout>

<Callout title="Title" type="warn">
  Hello World
</Callout>

<Callout title="Title" type="error">
  Hello World
</Callout>

  Hello World
</Callout>
