## 高频主线

- Fiber 解决了什么问题
- `useEffect`、`useLayoutEffect` 的差异
- 状态批处理和更新调度如何理解
- 如何判断一个组件需要优化，优化顺序是什么

## 回答建议

1. 先说 React 如何安排更新。
2. 再解释 Hooks 的运行约束。
3. 最后补充项目里遇到的性能瓶颈与排查方式。

## 常见误区

- 把记忆优化手段当成性能优化本身
- 没有区分渲染次数、计算开销和提交阶段的成本
