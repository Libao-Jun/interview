## 高频问题

- BFC 是什么，它解决了什么问题？
- Flex 和 Grid 各自适合什么场景？
- `position: sticky` 为什么会失效？
- 如何理解语义化标签与可访问性？

## 回答框架

### 1. 先说布局上下文

很多 CSS 问题都不是“某个属性背不下来”，而是没有先判断元素处在什么布局上下文里。

### 2. 再说约束条件

比如 sticky 失效，通常要检查滚动容器、父元素 overflow、以及元素可滚动空间是否成立。

### 3. 最后补实践经验

可以补充你在复杂表格、双栏布局、响应式页面中是如何选 Flex/Grid 的。

## 复盘提示

- 这类题不要只写定义，要写“现象 -> 原因 -> 排查点”。
- 如果能结合 DevTools 的 layout / computed 面板来描述，会更像真实工程经验。
