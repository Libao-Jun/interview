> 基础题通常不只是考记忆，而是在判断你的上限是否足够高。

## 这一章的目标

- 能说清浏览器如何把资源变成可交互页面
- 能解释 JavaScript 执行模型，而不是只背 API
- 能从布局、重排、重绘和网络链路推导优化思路

## 推荐学习路径

<Steps>

### HTML 与 CSS

语义化标签、盒模型、Flex/Grid 布局、选择器优先级、重排与重绘

### JavaScript 与 TypeScript

闭包、原型链、事件循环、类型系统、泛型与类型守卫

### 浏览器与网络

从 URL 到渲染全链路、HTTP 缓存策略、TCP/UDP、跨域解决方案

</Steps>
